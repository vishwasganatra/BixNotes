# BixNotes

**A tool to increase productivity in day-to-day life.**

## Features

- _Build with **Java**_
- _Custom DialogBoxes_
- _Custom Buttons and UI_
- _**Dark** theme UI_
- _Notes with **color** options_
- _Notes with **URL** & **images**_
- _Searching notes using custom EdiText_
